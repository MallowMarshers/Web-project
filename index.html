Seja bem vindo a minha página! Sou Diego Pereira e esta é minha página web. Esta página se destina a um projeto de faculdade para apliações da net, mas pode conferir o que venho desenvolvendo durante os meus estudos.

Eu desenhei essa página como inspiração de sites antigos dos anos 90 e 2000. Esses modelos foram base para a criação desta página, utilizei alguns recursos modernos do HTML 5 e CSS para dar um ar levemente moderno para o site, apesar de não conter implementações de script, o site será uma fundação para desenvolvimentos futuros, talvez até servirá como um site real de portifolio. Entretanto, esteja a vontade para dar uma espiada no meu site, você poderá ver algumas alterações com o tempo e futuramente terá uma cara completamente nova!
